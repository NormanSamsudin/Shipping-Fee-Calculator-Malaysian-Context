#the application can be run in docker

#here is the step so that we can run this in docker

1. Clone my github

2. Install docker-desktop(windows) or docker(linux)

3. Open project folder

4. Build docker: docker-compose -f docker-compose.yml build

5. Run docker: docker-compose -f docker-compose.yml up

6. Stop docker: docker-compose -f docker-compose.yml down
